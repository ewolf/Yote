Yote Hello World Example

This program contacts the GServ Server to get its hello world app.
The messages are in the following format
  • a : app class
  • c : command
  • d : argument list (optional)
  • t : login token
The message sent looks like 
 {
   a => 'GServ::Hello',
   c => 'hello',
   t => token
 }
However this does require a Yote account.
