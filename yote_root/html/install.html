Installing Yote

You can install Yote using either CPAN or Github.

CPAN
$ sudo CPAN Yote
Github
Get Yote on github : https://github.com/ewolf/Yote
$ perl Build.PL
$ ./Build installdeps
$ ./Build test
$ sudo ./Build install

Running Yote

Yote can be run on any available port. The yote_server script starts, stops or restarts yote.
$ yote_server --port=80 start|stop|restart

Yote will create a datastore if none exists, and use yote_root for any html it may serve.

Configuring Yote

  • Create Root Account
    • When Yote is started, the first account created is the root account.
    • Open up your browser to http://localhost:yote port/yote/index.html
    • click 'Log in' on the upper right side
    • Then click the 'create account' button
    • Fill out the form to create the account
    • This account will then be the root account and have root privileges.
  • Adding Classpaths
    • Stop the server useing $ yote_server stop
    • Run $ yote_classpath_updater.pl space separated directories
    • Yote will now be able to load apps contained in those directories
    • Restart the server useing $ yote_server --port=80 start
  • Run the javascript unit tests
    • Open up http://localhost:yote port/yote/unit_test.html
    • Confirm that the tests pass successfully
