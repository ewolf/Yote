Documentation

This documentation describes how Yote programs are written, and what idioms are used.

Client Side

Yote calls are made from the client side javascript, so this is a good place to start.

Yote javascript methods

There are 11 methods that are meant to be invoked when coding yote programs, but only a handful of those will be invoked commonly.

common methods :
init fetch_app
user methods :
create_login login logout is_logged_in get_login fetch_account remove_login
other methods :
upload fetch_root
